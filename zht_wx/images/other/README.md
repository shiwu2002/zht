## 图标获取方式

### 方式 1: 使用 iconfont.cn
1. 访问 https://www.iconfont.cn
2. 搜索需要的图标 (首页、分类、消息、用户)
3. 下载 PNG 格式，尺寸 81x81
4. 替换对应文件

### 方式 2: 使用 Figma/Sketch
1. 设计 81x81 的图标
2. 导出 PNG 格式
3. 未选中状态使用灰色 (#999999)
4. 选中状态使用绿色 (#07c160)

### 方式 3: 使用在线工具
- https://www.iconfinder.com
- https://www.flaticon.com
