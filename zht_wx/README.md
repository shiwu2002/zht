# 置换通 - 微信小程序

基于 Spring Boot 后端的物品交换平台微信小程序前端。

## 快速开始

### 1. 配置后端地址

修改 `utils/api.js` 中的 `BASE_URL`:

```javascript
const BASE_URL = 'http://your-backend-ip:8080/api';
```

### 2. 导入微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择此目录
3. 填写您的 AppID
4. 编译运行

### 3. 图标说明

项目已包含占位图标，可以正常运行。
如需替换正式图标，请查看 `images/README.md`。

## API 对接

所有 API 接口已封装在 `utils/api.js` 中：

| 模块 | 接口数 | 说明 |
|------|--------|------|
| userApi | 5 | 登录、信息、更新、绑定、信用分 |
| itemApi | 7 | 发布、更新、删除、详情、列表、我的、附近 |
| exchangeApi | 7 | 创建、确认、拒绝、取消、完成、详情、列表 |
| favoriteApi | 4 | 添加、取消、检查、列表 |
| messageApi | 5 | 发送、会话、历史、已读、未读数 |
| categoryApi | 2 | 列表、热门 |
| fileApi | 2 | 上传、删除 |
| reviewApi | 3 | 添加、用户列表、交换评价 |
| reportApi | 3 | 提交、列表、处理 |

## 注意事项

1. 确保后端服务已启动且可访问
2. 微信小程序需要配置合法域名
3. 图片上传需要后端支持
4. 地理位置功能需要用户授权

## 常见问题

### Q: tabBar 图标不显示？
A: 当前使用占位图标，可以正常运行。如需显示正式图标，请替换 `images/` 目录下的 PNG 文件。

### Q: 如何配置后端地址？
A: 修改 `utils/api.js` 第 2 行的 `BASE_URL`。

### Q: 登录失败？
A: 检查后端服务是否启动，网络是否通畅，AppID 是否正确。
