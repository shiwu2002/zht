## 📱 使用方式

现在用户点击"取货位置"时有三种选择：

### 方式一：自动获取（推荐）
1. 点击"取货位置"
2. 选择"自动获取"
3. 系统会请求位置权限
4. 用户同意后自动获取当前位置
5. 通过腾讯地图 API 转换为详细地址

### 方式二：地图选点
1. 点击"取货位置"
2. 选择"手动输入"
3. 点击"📍 地图选点"按钮
4. 在地图上选择任意位置
5. 自动填充该位置的详细信息

### 方式三：手动输入
1. 点击"取货位置"
2. 选择"手动输入"
3. 在输入框中填写详细地址
4. 点击"确定"保存

## 🔧 发布小程序时的配置

### 小程序后台配置
发布前需要在微信公众平台配置服务类目：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入"设置" → "基本设置" → "服务内容声明"
3. 添加服务类目：**出行服务 - 地图** 或 **工具 - 效率**
4. 填写服务范围和用途说明

### 隐私协议配置
在小程序后台配置用户隐私保护指引：

1. 进入"设置" → "基本设置" → "服务内容声明"
2. 找到"用户隐私保护指引"
3. 添加位置信息收集和使用场景说明：
   - **收集位置信息**：用于发布物品时标注物品所在位置
   - **使用场景**：方便其他用户了解物品位置，促进交易

## 🐛 常见问题

### Q1: getLocation 报错？
A: 确保已在 `app.json` 中配置 `requiredPrivateInfos`

### Q2: 逆地理编码失败？
A: 检查腾讯地图 Key 是否正确，是否有可用配额

### Q3: 定位不准确？
A: GPS 定位在室内可能不准，建议在室外测试

### Q4: 用户拒绝授权怎么办？
A: 引导用户到微信设置中手动开启位置权限
